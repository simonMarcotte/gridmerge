# GridMerge

GridMerge merges multiple PDFs into one customizable grid-style output.

## Access

Production website: https://d1x14g0h3qz9la.cloudfront.net

## Current Issues

- Merge time can vary a lot for large or image-heavy PDFs.
- Worker scaling is functional but still being tuned for cost versus latency.
- Grid layout in preview not centered or fully representative of final layout.
